Adaptive Coefficient of Variation Round Robin Scheduling

ACV-RR Algorithm

Algorithm Overview

ACV-RR (Adaptive Coefficient of Variation Round Robin) is a dynamic scheduling algorithm that adjusts time quantum based on process characteristics to minimize context switches while maintaining fairness.

Dynamic Quantum

Adjusts based on burst time and remaining time

Fair Scheduling

Ensures all processes get CPU time

Reduced Context Switches

Optimizes quantum to minimize overhead

Pseudocode

Algorithm: ACV-RR (Adaptive Coefficient of Variation Round Robin)

1. Initialize:
   CurrentTime ← 0
   ReadyQueue ← Empty
   RemainingBurstTime ← BurstTime of each process
   CompletedProcesses ← 0

2. While CompletedProcesses < TotalProcesses
   Add all arrived processes to ReadyQueue
   
   If ReadyQueue is empty
      CurrentTime ← CurrentTime + 1
      Continue
   
   Calculate:
      Mean (μ)
      Standard Deviation (σ)
      Coefficient of Variation (CV)
      MinBT ← Minimum Remaining Burst Time
      MaxBT ← Maximum Remaining Burst Time
   
   If CV < 0.20
      TQ ← (MinBT + MaxBT + μ) / 2
   Else If CV < 0.50
      TQ ← (MinBT + MaxBT + μ) / 3
   Else If CV < 0.70
      TQ ← (MaxBT × μ) / 3
   Else
      TQ ← (MinBT + μ) / 4
   
   TQ ← Ceiling(TQ)
   
   Sort ReadyQueue according to Remaining Burst Time (SJF)
   
   For each process Pi in ReadyQueue
      Execute Pi for:
         min(TQ, RemainingBurstTime(Pi))
      Update CurrentTime
      Update RemainingBurstTime(Pi)
      Add newly arrived processes
      
      If RemainingBurstTime(Pi) > 0
         Reinsert Pi into ReadyQueue
      Else
         Mark Pi as completed

3. For each process Pi
   Calculate Waiting Time:
      WaitingTime(Pi) = (Start Time of First Execution − Arrival Time)
                      + Σ(Start Time of Each Subsequent Execution − End Time of Previous Execution)
   
   Calculate Turnaround Time:
      TurnaroundTime(Pi) = WaitingTime(Pi) + BurstTime(Pi)

4. Calculate:
   Average Waiting Time
   Average Turnaround Time
   Context Switches

5. Return:
   Gantt Chart
   Waiting Times
   Turnaround Times
   Average WT
   Average TAT
   Context Switches

Key Advantages

  • Adaptive Quantum: Automatically adjusts based on remaining burst time
  • Efficient for Short Processes: Completes short processes quickly (≤4 time units)
  • Handles Long Processes: Uses larger quantum (up to 8) for long jobs
  • Reduced Overhead: Fewer context switches compared to fixed quantum RR
  • Maintains Fairness: Round-robin nature ensures no starvation
