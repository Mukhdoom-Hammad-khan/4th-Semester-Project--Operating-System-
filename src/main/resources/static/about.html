CPU Scheduling Algorithm Comparator

A web-based interactive tool that compares two CPU scheduling algorithms: ACV-RR (Adaptive Coefficient of Variation Round Robin) and Simple Round Robin. The project helps visualize how different algorithms perform with various process datasets.

An educational platform designed to demonstrate and compare CPU scheduling algorithm performance through interactive visualization.

Project Idea

The main idea is to create an educational platform where users can input process data and see real-time comparisons between algorithms. The system shows which algorithm performs better in terms of waiting time, turnaround time, and context switches through interactive charts and detailed calculations.
